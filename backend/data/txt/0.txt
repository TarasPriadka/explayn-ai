    Computer Security  By David Wagner, Nicholas Weaver, Peyrin Kao, Fuzail Shakir, Andrew Law, and Nicholas Ngai Additional contributions by Noura Alomar, Sheqi Zhang, and Shomil Jain This is the textbook for CS 161: Computer Security at UC Berkeley. It provides a brief survey over common topics in computer security including memory safety, cryptography, web security, and network security.   Corrections  As of the Fall 2021 semester, this textbook is still being actively maintained and updated. Please contact cs161-staff@berkeley.edu for information regarding corrections.
